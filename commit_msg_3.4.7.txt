🎨 RESPONSIVE DESIGN PATTERNS

**Pattern 1: Progressive Grid Column Reduction**
```css
/* Desktop: 4 columns */
.grid { grid-template-columns: repeat(4, 1fr); }

/* Tablet: 2 columns */
@media (max-width: 1024px) {
  .grid { grid-template-columns: repeat(2, 1fr); }
}

/* Mobile: 1 column */
@media (max-width: 768px) {
  .grid { grid-template-columns: 1fr; }
}
```

**Pattern 2: Horizontal Scroll Tables on Mobile**
```css
/* Desktop/Tablet: Full-width */
.table-container { overflow-x: visible; }

/* Mobile: Horizontal scroll */
@media (max-width: 768px) {
  .table-container { overflow-x: scroll; }
  .table { min-width: 600px; }
}
```

**Pattern 3: Form Layout Stacking**
```css
/* Desktop/Tablet: Horizontal */
.form-row { display: flex; gap: 1rem; }

/* Mobile: Stacked */
@media (max-width: 768px) {
  .form-row { flex-direction: column; }
  .form-row input { width: 100%; }
}
```

**Pattern 4: Split-View to Stacked**
```css
/* Desktop: Side-by-side */
.split-view { grid-template-columns: 1fr 1fr; }

/* Tablet/Mobile: Stacked */
@media (max-width: 1024px) {
  .split-view { grid-template-columns: 1fr; }
}
```

**Pattern 5: Card Grid Auto-Fill**
```css
/* Desktop: Auto-fill with min-width */
.card-grid {
  grid-template-columns: repeat(auto-fill, minmax(380px, 1fr));
}

/* Tablet: Explicit 2-column */
@media (max-width: 1024px) {
  .card-grid { grid-template-columns: repeat(2, 1fr); }
}

/* Mobile: Single column */
@media (max-width: 768px) {
  .card-grid { grid-template-columns: 1fr; }
}
```
